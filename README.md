# 사립대 사학재단 재정 추적 — 1차 자료 직접 추출

하네스 뉴스룸 탐사 보도 시리즈의 인터랙티브 대시보드. **한국사학진흥재단 대학재정알리미 통합공시 API를 직접 호출해 받은 4년제 일반대 191곳의 11년치 시계열**이 핵심이다.

## 데이터 출처

- **1차** · 한국사학진흥재단 대학재정알리미 (uniarlimi.kasfo.or.kr)
  - 엔드포인트: `POST /totalAnno/properties/property/search`
  - 항목: searchType 1(확보율) / 2(수익률) / 3(학교운영경비 부담률)
  - 범위: 4년제 일반대학(gradeDiv=10) 학교별 191곳, 2014~2024 회계연도
  - 인증 불필요. JSON 응답.
- **2차** · 일부 섹션에서 인용
  - 대학교육연구소 「2024년 사립대 적립금 분석」 — 적립금 11.6조 / 건축 46.3%
  - 국가법령정보센터 사립학교법·사학기관 재무회계규칙 — 제도 설명

## 핵심 발견 (자체 추출)

- 4년제 일반대 평균 수익용 기본재산 확보율: **2014년 47.8% → 2024년 98.3%**, 11년간 두 배 상승. 그러나 191곳 중 **102곳(53.4%)이 100% 미달**.
- 2024년 학교운영경비 부담률: 191곳 중 **97곳이 상한(125%)에 박혀** 있어 평균이 인위적으로 평탄. 7곳은 권고 부담이 있는데 실제 부담 0원.
- 자산 1,000억 이상 보유하면서 종합 수익률 0%대인 학교 15곳. 가장 극단적 사례 — 자산 약 2조원, 운용 수익 약 6억원.

## 파일 구성

- `index.html` — 대시보드 (단일 파일, Tailwind+Chart.js CDN, `data.json` 동적 로드)
- `data.json` — 정규화된 자체 추출 데이터
- `fetch_data.sh` — 대학재정알리미 API 호출 스크립트 (재현용)
- `raw_data/` — 원본 JSON 응답 (연도·항목별 28개 파일)
- `README.md` / `LICENSE` / `.nojekyll`

## 한계

- 수익용 기본재산 확보율의 산식 분모가 2024년 9월 「대학설립·운영 규정」 개정으로 '운영수익 총액' → '등록금·수강료 수입액'으로 축소됐다. 2024년 값은 산식 변경 효과를 포함.
- '학교운영경비 부담률'은 법인전입금 비율과 다르다. 부담률은 학교가 쓰는 운영비 중 법인이 부담한 권고 분담 대비 실제 부담. 법인전입금 비율은 운영수익 대비 법인전입금. 이번 라운드에선 부담률만 자체 추출.
- 회계 결산(univFinance/accounts) API는 acntDiv 동적 로드 필요, 후속 라운드.
